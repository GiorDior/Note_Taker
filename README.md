# Note Taker Application

This is a simple note-taking application that allows users to create, save, and delete notes. It provides a user-friendly interface for managing notes efficiently.

## Description

This application is designed to streamline the process of taking and managing notes. It follows a simple yet effective workflow:

1. **Landing Page**: Upon opening the application, users are presented with a landing page containing a link to the notes page.

2. **Notes Page**: Clicking on the link navigates users to the notes page, where existing notes are listed in the left-hand column, and empty fields are provided in the right-hand column to enter a new note title and its text.

3. **Saving Notes**: After entering a new note title and its text, users can save the note by clicking the "Save Note" button in the navigation. A "Clear Form" button is also available to clear the input fields if needed.

4. **Viewing and Editing Notes**: Clicking on an existing note in the list displays the note's details in the right-hand column. A "New Note" button appears in the navigation to create a new note.

## Features

## Installation

To install and run the Note Taker Application locally, follow these steps:

1. Clone this repository to your local machine using `git clone`.
2. Navigate to the project directory.
3. Install dependencies by running `npm install`.
4. Start the application by running `npm start`.
5. Open your web browser and visit `http://localhost:3000` to view the application.

## Usage

- Visit the deployed application to start taking notes.
- Click on the link to the notes page to begin.
- Enter a new note title and its text in the provided fields.
- Click "Save Note" to save the note.
- Click on an existing note to view or edit it.
- Use the "New Note" button to create a new note.
